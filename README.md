# sonic-js
This is a WIP (work in progress) version of Sonic the Hedgehog in Javascript and Html5 using the ImpactJS Game Engine.

Please feel free to contribute. 

Roadmap:
- Loop de Loops
- Title Screen
- Start of Level Screen
- Springs
- Enemies
- Items
- Game Over Screen
- Character Select
- Level Select
- Finish HUD
- Finish Level 1
